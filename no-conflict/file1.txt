*note for dont conflict git*
1) Realizar as alterações em uma branch específica (crie uma nova para seu componente) - (git checkout -b /nomeDaBranch/);
2) Atualizar com a upstream (git pull upstream master);
3) Realizar um /pull upstream/ dentro da master (/git pull upstream master/);
4) Após o procedimento acima, commitar na branch recém criada (/git add/, /git commit -m "nomeDoCommit"/), (/git push origin /nomeDaBranch/);
5) Abrir o PR(Pull Request)
6) Mergear.

GG!
